# selfkey-vendor-manager
Tool to manage vendor on selfkey main contract


## How to run

- Replace the constants in the top with the corret values:

```javascript
const chainURL = 'https://ropsten.infura.io/v3/...';
const contractAddress = '...';
const privateKey = '...';
const vendorDID = '...';
```

- Run

```javascript
node index.js
```
